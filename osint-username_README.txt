osint-username
==============

وصف:
-----
هذه أداة مفتوحة المصدر للبحث عن اسم مستخدم (username) عبر أشهر مواقع التواصل الاجتماعي ومواقع الدردشة، بالإضافة إلى اقتراح والتحقق من البريد الإلكتروني بناءً على اسم المستخدم.

طريقة التثبيت:
--------------
1. تأكد أن لديك Python 3.7 أو أحدث.
2. ثبّت مكتبة requests عبر الأمر التالي:
   ```
   pip install requests
   ```
3. قم باستخراج ملفات الأداة في مجلد باسم osint-username، بحيث يكون لديك الملفات التالية على الأقل:
   - main.py
   - platforms.py
   - email_check.py

طريقة التشغيل:
--------------
1. افتح سطر الأوامر (Terminal).
2. انتقل إلى مجلد الأداة:
   ```
   cd osint-username
   ```
3. شغّل الأداة بالأمر:
   ```
   python main.py
   ```
4. سيطلب منك إدخال اسم المستخدم الذي تريد البحث عنه.
5. ستظهر لك النتائج لجميع المواقع المدعومة.
6. عند ظهور خيار البريد الإلكتروني، يمكنك إدخال مفتاح Hunter.io API إذا توفر لديك، أو اضغط Enter لفحص الصياغة فقط.

ملاحظات:
--------
- يمكنك إضافة أو تعديل مواقع التواصل والدردشة عن طريق ملف platforms.py.
- بعض المواقع قد لا تظهر نتيجة دقيقة إذا كان الحساب خاص أو يتطلب تسجيل دخول.
- للتحقق الفعلي للبريد الإلكتروني يجب أن تمتلك مفتاح API من موقع [hunter.io](https://hunter.io).
- الأداة قابلة للتطوير ويمكنك إضافة دعم التصدير أو مميزات إضافية بسهولة.

بالتوفيق!
