شرح تثبيت الأداة وتشغيلها بالأوامر البرمجية (الأكواد):

## 1. تثبيت متطلبات الأداة

تأكد أنك تملك Python 3.7 أو أحدث مثبت على جهازك.

قم بتثبيت مكتبة requests (إذا لم تكن مثبتة مسبقاً):
```bash
pip install requests
```

---

## 2. تحميل ملفات الأداة

- ضع الملفات التالية في مجلد باسم `osint-username`:
    - `main.py`
    - `platforms.py`
    - (وأي ملفات أخرى تابعة للأداة مثل `email_check.py`)

---

## 3. تشغيل الأداة

انتقل إلى مجلد الأداة عبر الطرفية (Terminal):
```bash
cd osint-username
```

لتشغيل الأداة اكتب:
```bash
python main.py
```

---

## 4. استخدام الأداة

- عند تشغيل الأداة سيظهر لك ترحيب باسم المبرمج والمعلومات.
- أدخل اسم المستخدم الذي ترغب في البحث عنه عند الطلب.
- ستظهر لك النتائج لجميع المواقع المدعومة.
- يمكنك إدخال مفتاح Hunter.io لفحص البريد الإلكتروني أو الضغط Enter لتخطي الفحص المتقدم.

---

**ملاحظات:**
- يمكنك تعديل ملف `platforms.py` لإضافة أو حذف مواقع حسب رغبتك.
- بعض المواقع قد لا تعطي نتائج دقيقة إن كان الحساب خاص أو يتطلب تسجيل دخول.
- إذا ظهرت مشكلة في تشغيل الأداة، تأكد من تثبيت جميع المتطلبات وأنك تستخدم نسخة Python حديثة.
